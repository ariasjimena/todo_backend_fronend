#instrucciones

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

start the database

```bash 
  mongod
```

Start the server

```bash
  npm run dev
```

start the frontend

```bash
  npm start
```
